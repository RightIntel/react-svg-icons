# react-svg-icons

React components for svg icons

A license is required to use Lini icons

## Usage

```jsx harmony
import React from 'react';
import { LiniAlarm } from 'react-svg-icons';

export function MyComponent() {
    return (
        <div className="MyComponent">
            <LiniAlarm size={24} />
        </div>
    );
}
```

## Changing Color

```css
.MyComponent .Lini {
    fill: orangered;
}
```
